About

Task Force: Eclipse is an original, COD‑inspired tactical prototype. Every mission is a fresh run: find the Intel, manage noise, and extract through the green zone.

How to Play

Press New Operation in the game to generate a new map. Pick up the blue Intel, then return to the green extraction zone. Enemies react to sound and line‑of‑sight.

Tech

Pure HTML5 Canvas + vanilla JS. Zero external assets or libraries. Works offline once loaded.
